How to use

  • Search: Type in the search box to filter the table. You can search for decimal, hex, octal values, or the actual character.
  • Navigate: Use arrow keys to move between cells within a table. Home/End to jump to first/last cell in current table.
  • Switch tables: Use Ctrl+Right Arrow and Ctrl+Left Arrow to move between the four table sections.
  • Select: Double-click or press Enter/ Space on a cell to copy its value and add to output area.
  • Quick search: Press Ctrl+F to focus the search box.
  • Clear search: Press Esc in the search box to clear it.
  • Output area: Double-click the output area to copy all accumulated values.
