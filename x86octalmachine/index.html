1. 8086 Baseline

Start with Intel’s original programmer’s model

Before the octal thesis, Intel’s 1978-1979 documentation defines a 16-bit CPU with a 20-bit physical address space, four segment registers, a 16-bit instruction pointer, and nine visible flags.

Programmer’s Model

Registers, segments, IP, and FLAGS

The original 8086 model is not just an opcode table. It is a register file, a segmented memory model, and a set of control and status flags that shape how instructions behave.

Address Formation

20-bit physical addresses from segment:offset pairs

Intel describes the 8086 as a 16-bit processor that reaches a one-megabyte physical space by combining a segment base with an offset.

Physical Address Rule

physical = (segment << 4) + offset

Segments begin on 16-byte boundaries. A segment register supplies the base, then a 16-bit offset selects a location inside that 64 KB window.

Key Constraints

  • 1 MB physical address space
  • 64 KB maximum per segment
  • CS, DS, SS, and ES are the active segment registers
  • The hardware stack is word-oriented and grows downward in memory

Intel 1979 Scope

What the original spec actually centers on

The 8086 manuals cover more than encodings: the execution unit, bus interface unit, segmentation, stack discipline, interrupts, I/O, and the full instruction-set families.

Architecture
Execution unit, bus interface unit, registers, flags, and instruction pointer.
Memory Model
Segmentation, physical address generation, stack implementation, and reserved memory locations.
Control Flow
Interrupts, program transfer, flag operations, synchronization, and processor control.

2. Octal Atlas

Read the map before the table

Every cell is one byte from 000 to 377. Start with the four valid byte ranges, then drill down into individual opcodes inside each range.

Byte Ranges

Four real bands in one byte

A byte in octal stops at 377, so the atlas is split into four top-level ranges: 000–077, 100–177, 200–277, and 300–377.

Opcode Field

Read it as four octal boards

Each board fixes the first digit. Rows are the middle octal digit, columns are the last one, and selected anchors get a short tag instead of a full caption block.
